🚀 TesteHTML Publisher

Cole seu código HTML e publique diretamente no GitHub

⚠️ Importante: Este app usa a API do GitHub diretamente do navegador. O token está pré-configurado para facilitar testes e tem permissão limitada!

🔧 Configuração do GitHub

📋 Como usar:

  1. Digite um nome para seu arquivo HTML
  2. Cole seu código HTML completo
  3. Clique em "Publicar no GitHub"
  4. Acesse via: tehew.space/testehtml/nome-arquivo.html
Será salvo como: arquivo.html
