💡 Como funciona?

  • ➕ Criar Usuário: Clique no botão ➕ para criar um novo perfil
  • 🔄 Trocar Usuário: Selecione no dropdown para alternar entre usuários
  • ✏️ Renomear Usuário: Clique no botão ✏️ para alterar o nome do usuário ativo
  • 🗑️ Deletar Usuário: Clique no botão 🗑️ para remover permanentemente (cuidado!)
  • 📊 Dados Separados: Cada usuário tem suas próprias transações
  • ☁️ Sincronização: Dados salvos automaticamente na nuvem

✨ Exemplos de Uso

  • Pessoal: "João", "Maria" - finanças individuais
  • Familiar: "Pai", "Mãe", "Família" - gastos separados e conjuntos
  • Profissional: "Pessoal", "Trabalho", "Empresa"
  • Projetos: "Projeto_A", "Projeto_B", "Freelance"
